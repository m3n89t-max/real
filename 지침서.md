## 2. 전략 아키텍처

### 2.1 타임프레임 계층 (5단계)

```
4h  ─→ 최상위 추세 방향 (Big Degree)
1h  ─→ 중기 파동 방향 확인
15m ─→ 세부 파동 필터
5m  ─→ 메인 매매 로직 (진입 구간 판단, ATR/볼륨 계산)
1m  ─→ 정밀 진입 타점
```

### 2.2 진입 필터 체인 (순서대로 통과 필수)

1. **4h+1h 추세**: 모두 횡보면 스킵
2. **5m 파동 분석**: 진입 구간 + 절대법칙 통과
3. **4h 방향 일치**: 명확히 반대면 스킵
4. **볼륨 확인** (v3.0 신규): 5m 거래량 ≥ 20봉 평균
5. **EMA 정렬** (v3.0 신규): 9/21 EMA 방향 일치
6. **RSI 다이버전스**: 역방향 + 낮은 신뢰도면 스킵
7. **손익비**: 최소 1:2.0 (기존 1:1.5)
8. **총 노출도** (v3.0 신규): 계좌의 50% 이내

## 3. 리스크 관리 (v3.0)

### 3.1 포지션 크기

```
최대 허용 손실 = 계좌 × 1.5% (기존 2%)
명목금액      = 허용 손실 / SL거리(%)
수량          = 명목금액 / 진입가
```

### 3.2 동적 손절 (ATR 기반)

```
SL거리 = max(ATR% × 2.0, 기본SL 2%)
하한   = 1.5% (너무 타이트 방지 → 스탑헌팅 방어)
상한   = 5.0% (과도한 손실 방지)
```

| 변동성 | ATR% | 실제 SL% |
|--------|------|----------|
| 낮음 | 0.5% | 1.5% (하한) |
| 보통 | 1.2% | 2.4% |
| 높음 | 2.0% | 4.0% |
| 극한 | 3.0%+ | 5.0% (상한) |

### 3.3 레버리지 (하향 조정)

| 조건 | v2.0 | v3.0 | 근거 |
|------|------|------|------|
| 파동 단독 | 4x | **3x** | 리스크 감소 |
| PRG 또는 RSI | 7x | **5x** | 중간 확신 |
| PRG + RSI 동시 | 10x | **7x** | 최고 확신 |

### 3.4 동시 포지션

- 최대 **3개** (기존 4개)
- 총 명목가 합계 ≤ 계좌의 **50%**

## 4. 수익 보호 시스템 (v3.0 신규)

### 4.1 본전이동

```
수익이 SL거리의 0.5배 이상 → SL을 진입가로 이동
(예: SL=3%, 현재 수익 +1.5% → 손절가를 진입가로 올림)
```

### 4.2 트레일링 스탑

```
수익이 SL거리의 1.0배 이상 → 트레일링 활성화
트레일링 거리 = ATR × 1.5
(가격이 상승하면 SL도 따라 올라감)
```

### 4.3 분할 익절

```
TP1: 포지션의 50% → 손익비 1.5배 도달 시 실현
TP2: 나머지 50%   → 트레일링 스탑으로 관리 (최대 RR 3.0까지)
```

**효과**: 확실한 수익을 먼저 확보 + 나머지는 추세 이탈 시까지 보유

## 6. 파동 분석 절대법칙 (변경 없음)

### 임펄스

1. **3파 최소 법칙**: 3파가 가장 짧으면 즉시 기각
2. **2파 되돌림**: 1파의 61.8% 절대 초과 금지
3. **연장의 법칙**: 가장 긴 파동 ≥ 두 번째 × 1.618
4. **4파 비겹침**: 4파 끝이 1파 시작점을 침범 불가

### 조정파 (지그재그/플랫)

- **지그재그 B파**: A파의 61.8% 이하 되돌림
- **플랫 B파**: A파의 61.8% 이상 되돌림
- **C파 목표가**: 피보나치 비율 기반
